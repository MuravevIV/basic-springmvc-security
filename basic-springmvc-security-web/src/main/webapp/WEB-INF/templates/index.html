Basic Spring MVC + Spring Security

Features

  1. login/password form authentication
  2. remember-me (persistent-login) authentication
  3. page authorization
  4. password protection
  5. CSRF protection

Demo

  1. Login as user/user - you will be redirected back here. You are now authenticated as user.
  2. Logout - you will be logged out and be suggested to sign in again.
  3. Login as user/user and try to access admin console page - "Access is denied" (403) error.
  4. Login as admin/admin, then try to access admin console page - access granted.
  5. Close / open browser application, then try to access admin console - application doesn't remember you and suggests to sign in.
  6. On the login page check "Remember me" checkbox, then login as admin/admin. Close / open browser application, then try to access admin console - application remembers you.
